Usage

1. It is necessary you have defined on Matlab the X - data matrix. Size of matrix (X)
must be n-by-p; n=observations, p-independent variables=columns. 

2. For running this file it is necessary to call the sphertest function as Mausphercnst(X,alpha)
(alpha-significance level default = 0.05). Please see the help sphertest.

3. Once you input your information, it will appears your results.
